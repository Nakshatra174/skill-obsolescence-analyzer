About the Project

The Skill Obsolescence Prediction and Comparative Analysis System is a data-driven web application designed to analyze the longevity of technical skills in a rapidly evolving job market.

Problem Statement

With increasing automation and technological advancements, many skills lose relevance over time. Individuals often lack tools to assess whether a skill is worth investing in for long-term career growth.

Proposed Solution

This system evaluates skills using structured datasets and weighted scoring techniques to estimate obsolescence risk. It also allows comparison between skills to support informed decision-making.

Technology Stack

  • Frontend: HTML, CSS
  • Backend: Python (Flask)
  • Dataset: CSV-based skill metrics
  • Analysis: Rule-based weighted scoring

Future Scope

  • Graphical trend analysis
  • Skill recommendations
  • Admin dataset management
  • Explainable AI insights
